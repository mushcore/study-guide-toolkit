KEY DISTINCTIONS TO MEMORIZE
=============================================================================

SEMANTIC KERNEL vs. MAF:
  SK:  Prompt orchestration, plugin composition, skill management
  MAF: Multi-turn reasoning, tool calling, agent autonomy

OLLAMA vs. GITHUB MODELS:
  Ollama:        Local, offline, free, limited model choice
  GitHub Models: Cloud, requires auth, broader model options, latency

STDIO vs. SSE (MCP):
  stdio: Synchronous, process-spawned, low overhead
  SSE:   Asynchronous, HTTP-based, scalable

ChatHistory vs. AgentSession:
  ChatHistory:  Manual list of ChatMessage objects
  AgentSession: Managed by agent, includes hidden state
