Checklist: Glossary
This checklist helps make sure that all entries of a Glossary are well-formed and legitimate.
Relationships
Related Elements

Main Description
Check Items
Does each term have a clear and concise definition
Is each glossary term included somewhere in the requirements work products
If not, it may imply that a requirement is missing or that the existing requirements are not complete. It is more likely, though, that the term is not included because it is not needed. In that case, you should remove it.
Are terms used consistently in the requirements work products
Does a term represent the same thing in all requirements work products
