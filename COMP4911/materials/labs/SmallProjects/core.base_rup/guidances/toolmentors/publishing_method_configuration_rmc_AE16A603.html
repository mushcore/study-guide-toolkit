Tool Mentor: Publishing a Method Configuration Using Rational Method Composer
This tool mentor describes how to publish a Method Configuration as a web site using Rational Method Composer.
Tool: Rational Method Composer

Main Description

A Method Configuration is a user-defined subset of the Method Library contained in Rational Method Composer (RMC).

Before it can be published, a configuration must be created.  For information on how to create a method configuration, see Tool Mentor: Creating a Method Configuration Using Rational Method Composer.

To publish an existing configuration, select "Configuration>Publish" and follow the prompts.  For more information on publishing a method configuration using RMC, see the "Publishing a Method Configuration to a Method Web" online help topic.  All RMC online help can be accessed under the Help menu.
