Main Description

There are two kinds of process in Rational Method Composer (RMC) -- Capability Patterns and Delivery Processes. 

Before it can be published, a process must be created.  For information on how to create a process, see Tool Mentor: Developing Processes Using Rational Method Composer.

Once created, processes may be exported from RMC to a planning tool where they can be used to manage a project.

To export an existing process, select "File>Export", select "Project Template" and follow the prompts.  For more information on exporting a process using RMC, see the "Exporting Processes and Method Content to a Planning Tool" online help topic.  All RMC online help can be accessed under the Help menu.
