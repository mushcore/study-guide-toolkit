Tool Mentor: Developing Method Content Using Rational Method Composer
This tool mentor describes how to develop method content elements such as roles, tasks, work products, and guidance. This tool mentor describes how to create new method content, tailor existing method content, and replace existing method content.
Tool: Rational Method Composer
Relationships
Related Elements
  • Tailor the Development Process for the Project
Main Description

Developing method content involves the creation and detailing of the method content elements (Role, Task, Work Product, and Guidance), as well as their organization into content packages.  It also involves the creation of Categories and Views. 

All method content must exist within a method plug-in.  For information on how to create a method plug-in, see Tool Mentor: Creating a Method Plug-In Using Rational Method Composer.

To decide where to place the element to be created or to locate the element to be tailored or replaced, you may need to navigate existing method content.  For information on navigating the method content, see Tool Mentor: Navigating Method Content Using Rational Method Composer.

For information on developing method elements, see the following cheat sheets: "Add New Method Content", "Tailor Existing Method Content", "Replace Existing Method Content", and "Add a Method Reference or Hyperlink". All Rational Method Composer cheat sheets can be accessed by selecting Help>Cheat Sheet.
