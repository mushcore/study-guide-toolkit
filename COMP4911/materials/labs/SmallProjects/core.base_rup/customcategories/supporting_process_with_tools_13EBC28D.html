Main Description

Many activities in the Rational Unified Process are supported by software-engineering tools.

Tool Mentors provide detailed descriptions of how to perform specific process activities or steps, or produce a particular artifact or report, using one or more tools. Some Rational tools appear in different versions of Rational Suite products. For a list of these particular details, visit our IBM/Rational Product Home Page, which requires Internet access.
