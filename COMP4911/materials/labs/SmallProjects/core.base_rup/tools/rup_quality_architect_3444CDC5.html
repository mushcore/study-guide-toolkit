Main Description

IBM® Rational® QualityArchitect application is a powerful system testing extension to Rational Rose®. It automatically generates test drivers and stubs from UML models to support functional and performance testing of EJB and DCOM/COM+ components and subsystems. Rational QualityArchitect improves software robustness and predictability by validating software quality early in the life cycle, instead of deferring it to system testing.
