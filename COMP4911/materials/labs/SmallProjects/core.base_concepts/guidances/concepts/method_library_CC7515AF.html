Concept: Method Library
A Method Library is a physical container for Method Plug-ins and Method Configuration definitions. All Method Elements are stored in a Method Library.
Main Description
Method Libraries represent physical storage of all Content and Process Elements as well as Method Configurations. A Method Library defines a closed universe for all elements in it, since  library elements cannot refer to other libraries.  All user-defined extensions to method content and processes have to be done with Method Plugins within the same physical library.
