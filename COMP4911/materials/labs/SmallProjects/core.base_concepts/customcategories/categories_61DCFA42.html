Main Description

Categories provide different ways to logically organize Content Elements. In addition to Standard Categories that have been predefined for most Content Element types in UMA, Custom Categories can be used to either of the following:

  • Categorize content based on the user's criteria.
  • Define tree-structures of nested categories allowing the user to systematically navigate and browse Method Content as well as Processes based on these categories. When Custom Categories are used in this way they are also referred to as Views.
