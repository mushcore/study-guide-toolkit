#### Test levels

-   **Unit** — individual components in isolation (white-box)
-   **Integration** — component interactions
-   **System** — end-to-end, full behavior (black-box)
-   **Acceptance** — user validation against requirements (black-box — Quiz 4 Q9)

#### Test types (Quiz 4 tested)

-   **Performance** — response time + processing time (Quiz 4 Q6)
-   **Load** — concurrent users / throughput
-   **Volume** — large data
-   **Configuration** — different environments / hardware / OS

#### White-box vs Black-box

-   White-box: knows internal code (unit tests)
-   Black-box: tests I/O behavior only (system, acceptance)
