#### The acronyms

| Term | Meaning |
| --- | --- |
| **BCWS** / PV | Budgeted Cost of Work Scheduled — planned value through date |
| **BCWP** / EV | Budgeted Cost of Work Performed — earned value |
| **ACWP** / AC | Actual Cost of Work Performed |
| **BAC** | Budget At Completion — total planned budget |
| **EAC** | Estimate At Completion — projected final cost |

#### Formulas

-   **Cost Variance**: `CV = BCWP − ACWP`    (positive = under budget)
-   **Schedule Variance**: `SV = BCWP − BCWS`    (positive = ahead)
-   **Cost Performance Index**: `CPI = BCWP / ACWP`    (>1 = efficient)
-   **Schedule Performance Index**: `SPI = BCWP / BCWS`    (>1 = ahead)
-   **Estimate At Completion**: `EAC = BAC / CPI`

*Note: some sources use CV = ACWP − BCWP (reversed convention). Stick with the convention in the course slides: CV = BCWP − ACWP.*
