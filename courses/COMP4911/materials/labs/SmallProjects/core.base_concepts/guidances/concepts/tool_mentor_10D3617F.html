Concept: Tool Mentor
This Guidance Type shows how to use a specific tool to create part of a Work Product either in the context of or independently from a Task or Activity.
Main Description

Tasks, Steps, and associated guidelines provide general guidance to the practitioner. To go one step further, Tool Mentors are an additional means of providing guidance by showing how to achieve certain goals with a specific software tool. Tool mentors link Tasks with tools such as Rational Rose, Rational RequisitePro, Rational ClearCase, Rational ClearQuest, Rational Suite TestStudio. Tool Mentors almost completely encapsulate the dependencies of the content on the tool set, keeping the tasks free from tool details. An organization can extend the concept of Tool Mentor to provide guidance for other tools.
