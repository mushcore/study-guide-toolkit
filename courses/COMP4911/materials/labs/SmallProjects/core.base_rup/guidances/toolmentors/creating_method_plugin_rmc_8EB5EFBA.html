Main Description

A Method Plugin is used to package any changes (additions, deletions, extensions, etc.) to existing method content and processes. 

For information on how to create a method plug-in, see the "Create Your Own Method Plug-In" cheat sheet.  All Rational Method Composer cheat sheets can be accessed by selecting Help>Cheat Sheet.
