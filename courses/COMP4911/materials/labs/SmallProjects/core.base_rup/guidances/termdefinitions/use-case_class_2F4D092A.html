Term Definition: use-case (class)
Main Description
A description of system behavior, in terms of sequences of actions. A use case should yield an observable result of value to an actor. A use case contains all flows of events related to producing the "observable result of value", including alternate and exception flows.
More formally, a use case defines a set of use-case instances or scenarios.
