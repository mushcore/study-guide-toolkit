Notes on References
Describes the current and future plans for adding references via plug-ins.
Main Description

References can be contributed under the main RUP references pages by plug-ins.

In the future, references will be treated similarly to term definitions - plug-ins will be able to add their own references, and the references page will be generated, just like the glossary is today.
