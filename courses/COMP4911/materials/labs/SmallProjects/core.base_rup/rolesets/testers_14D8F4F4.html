Role Set: Testers
This role set is primarily involved in testing the software.
Relationships
Roles
  • Test Analyst
  • Test Designer

Main Description
 This role set deals with the specific skills unique to testing. Note that there are additional roles involved in the Test discipline that build on and extend the base skills of other role sets. These additional roles can be found in the other role sets arranged by the base skill-set they extend (i.e. Manager, Designer, Analyst).
