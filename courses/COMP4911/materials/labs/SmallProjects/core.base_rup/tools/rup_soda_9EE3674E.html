Main Description

The IBM® Rational® SoDA® application provides automatic generation of software documentation. Rational SoDA templates support Microsoft® Word® 97, 2000, and 2002 (XP). SoDA templates also support Adobe® FrameMaker®+SGML 5.5.6 on UNIX and Windows, and FrameMaker+SGML 6.0 on UNIX only.
