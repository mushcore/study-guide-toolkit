Create a file named staticwebapp.config.json

Place the file in the public folder

Add the following content to it:

{
    "navigationFallback": {
       "rewrite": "index.html",
       "exclude": ["/static/media/*.{png,jpg,jpeg,gif,bmp}",   "/static/css/*"]
    },
    "mimeTypes": {
        ".json": "text/json"
    }
}
