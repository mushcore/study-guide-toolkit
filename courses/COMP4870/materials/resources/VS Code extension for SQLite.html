Install the following VS Code extension:

 Once you have installed the extension, right click on the SQLite file and select “Open Database”.

Expand SQLITE EXPLORER:

To view as much data as possible close all tabs. Right-click on the table you are interested in and select “Show Table”.

Data should display in a new tab.

To close the database, right-click on the database in “SQLITE EXPLORER” and select “Close Database”.
