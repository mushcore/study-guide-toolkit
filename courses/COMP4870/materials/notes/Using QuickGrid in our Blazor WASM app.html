QuickGrid documentation:

https://aspnet.github.io/quickgridsamples/

1) Add this package to your Blazor app:

Microsoft.AspNetCore.Components.QuickGrid

2) Add this to your _imports.razor

@using Microsoft.AspNetCore.Components.QuickGrid

3) Change Students[] students declaration in Students.razor file to:

IQueryable<Student>? students;

What we have done here is convert the students object into an IQueryable object

4) In your Students.razor components page, replace the current load() method with this:

protected async Task load() {
    var studentArray = await httpClient.GetFromJsonAsync<Student[]>($"{Constants.BASE_URL}api/students");
    students = studentArray?.AsQueryable();
}

5) Replace the entire table with this QuickGrid component:

<QuickGrid Items="@(students)">
    <PropertyColumn Property="@(p => p.StudentId)" Sortable="true" />
    <PropertyColumn Property="@(p => p.FirstName)" Sortable="true" />
    <PropertyColumn Property="@(p => p.LastName)" Sortable="true" />
</QuickGrid>
