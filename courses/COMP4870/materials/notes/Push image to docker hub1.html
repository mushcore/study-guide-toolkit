In the following instructions assume that username on Docker Hub is snoopy and that you are creating a repository in Docker Hub named toon.

Replace snoopy with your Docker Hub username.

dotnet publish -o distrib

Create Dockerfile with the following content:

FROM mcr.microsoft.com/dotnet/aspnet:8.0
COPY distrib /app
WORKDIR /app
ENV ASPNETCORE_URLS http://+:80
EXPOSE 80/tcp
EXPOSE 443/tcp
ENTRYPOINT ["dotnet", "MyDotnetWebApp.dll"]

docker build --tag snoopy/toon:1.0.0 .

docker run -d -p 8888:80 snoopy/toon:1.0.0

          http://localhost:8888

docker login --username=snoopy

docker push snoopy/toon:1.0.0

docker run -d -p 8888:80 snoopy/toon:1.0.0
