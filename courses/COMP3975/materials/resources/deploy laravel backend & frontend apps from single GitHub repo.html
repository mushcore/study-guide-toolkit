This concerns the possiblity of keeping your solution (comprising backend and frontend code) in a single GitHub repo. I suggest you look at this GitHub repo that we started in class:

https://github.com/medhatelmasry/COMP3975PROJECT

It consists of two folders. Folder laravel-backend has the Laravel app and react-frontend simply has an index.html page that calls the Laravel API.

There are two workflow files:

1) main_comp3975prj.yml was created by Azure "App Services" and I made some edits to it so that the correct folder is chosen.

2) azure-static-web-apps-polite-sky-039219d1e.yml was created for the frontend by Azure "Static Web Apps" and no edits were made to it.

The two deployments on azure are:

Backend: https://comp3975prj.azurewebsites.net/api/students
Frontend: https://polite-sky-039219d1e.5.azurestaticapps.net/
